# Store Order Profit Prediction

## 🔍 Project Overview
This project analyzes store order data to identify key factors affecting profitability.

## Tools Used
- Power BI (Data Visualization)
- SPSS Modeler (Predictive Modeling)
- Excel (Dataset)

## Key Features
- Interactive dashboard for sales and profit analysis
- Region-wise performance insights
- Predictive model to classify Profit/Loss

## Dataset
Includes:
- Orders
- Order Details
- Customers
- Derived Fields

## Key Insights
- High discounts reduce profitability
- Some regions perform better than others
- Optimized pricing improves business performance

## Files Included
- Dataset (Excel)
- Power BI Dashboard
- Project Report (PDF)
- Presentation (PPT)
